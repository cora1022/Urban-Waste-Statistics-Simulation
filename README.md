# 도시 폐기물 통계 시뮬레이션

임시 CSV 생성용 합성 데이터 시뮬레이션입니다.

- 단위: kg/일
- 기준: 환경부 제6차 전국폐기물통계조사 생활폐기물 1인당 0.9506kg/일을 주거 시설 기준값으로 사용
- 계산: 거주 인구와 종사 인구 × 업종별 1인 1일 생활폐기물 계수 + 유동 인구 × 방문 배출계수 + 건설·의료·사업장 특수 폐기물 보정
- 설정: 건물 개수, 인구 기준 배율, 건물 유형 비율을 조정하면서 재구성 전 예상 거주/종사/유동 인구를 확인할 수 있음
- 도시 데이터 입력: 목표 거주 인구, 선택적 종사/유동 인구, 목표 폐기물 톤/일을 넣어 인구·폐기물 배율을 자동 계산할 수 있음
- 인구 보정: 화면에 배치된 건물은 실제 건물/블록을 대표하며, 재구성 후 거주/종사/유동 인구 합계가 설정창 예상치와 맞도록 보정됨
- CSV/툴팁: 종량제봉투 대상 폐기물, 음식물류 폐기물, 재활용가능자원, 대형폐기물 등 배출 카테고리와 비닐류, 음식물쓰레기, 건설폐기물 등 세부 폐기물로 분해하여 표시/내보냄

정책·시설 설계용 정밀 예측이 아니라, 알고리즘 수업/프로토타입에서 현실적인 크기의 임시 입력 데이터로 쓰기 위한 모델입니다.
